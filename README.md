# cnx-fullstack-deploy

# prerequisite
node v14 +

# How to run this app locally
1. in cnx-backend: npm install
2. in cnx-frontend: npm install
3. in cnx-frontend: npm run build
4. in cnx-backend: npm run start

then access: http://localhost:5000/
